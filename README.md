# Agentic AI in Cybersecurity

A demo project exploring agentic AI applications in cybersecurity.

## Goals

- Detect and respond to security threats autonomously
- Build intelligent threat analysis pipelines
- Create AI-powered security monitoring systems

## Folder Structure

```
├── data/           # Sample datasets, threat logs
├── notebooks/      # Jupyter notebooks for exploration
├── src/            # Source code
├── models/         # Trained models, model artifacts
├── tests/          # Unit and integration tests
└── docs/           # Documentation
```

## Getting Started

```bash
# Install dependencies
pip install -r requirements.txt

# Run tests
pytest tests/

# Explore notebooks
jupyter notebook notebooks/
```
